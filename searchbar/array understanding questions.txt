

1. **Sum of Array Elements**: Write a function that calculates the sum of all the elements in an array.

2. **Find Maximum Value**: Write a function that finds the maximum value in an array of numbers.

3. **Filter Even Numbers**: Write a function that filters out all even numbers from an array and returns a new array with only the even numbers.

4. **Count Occurrences**: Write a function that counts the number of times a specific element appears in an array.

5. **Reverse an Array**: Write a function that reverses the elements in an array without using the built-in `reverse()` method.

6. **Flatten an Array**: Write a function that flattens a nested array (an array of arrays) into a single array.

7. **Remove Duplicates**: Write a function that removes duplicate elements from an array while preserving the original order.

8. **Find Common Elements**: Write a function that finds and returns the common elements between two arrays.

9. **Rotate Array**: Write a function that rotates the elements of an array to the right by a given number of positions.

10. **Two Sum Problem**: Given an array of numbers and a target value, write a function that returns an array of two numbers from the input array that add up to the target.

11. **Sort Without Built-in Methods**: Write a sorting function (e.g., bubble sort or quicksort) to sort an array of numbers without using JavaScript's built-in sorting methods.

12. **Array Chunking**: Write a function that divides an array into smaller arrays of a specified size.

13. **Array Intersection**: Write a function that returns the elements common to multiple arrays.

14. **Array Difference**: Write a function that returns the elements that are unique to each array among multiple arrays.

15. **Map and Transform**: Write a function that takes an array and a transformation function and returns a new array with the transformed elements.

16. **Array Sum Pairs**: Given an array of numbers and a target sum, find all pairs of numbers in the array that add up to the target.

17. **Array Frequency Count**: Write a function that counts the frequency of each element in an array and returns an object with the counts.

18. **Array Shuffle**: Write a function to shuffle the elements of an array randomly.

19. **Subarray with Given Sum**: Given an array of numbers and a target sum, find a contiguous subarray that adds up to the target.

20. **Consecutive Array Elements**: Write a function that checks if an array contains consecutive numbers (in any order).
